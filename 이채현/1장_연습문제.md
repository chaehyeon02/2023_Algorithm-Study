### 1. n개의 수로 구성된 리스트（또는 배열）에서 가장 큰 수를 찾는 알고리즘을 작성하시오.

```C++
int findMax(int len, int arr[]) {
	int max = arr[0];
	for(int i = 1; i < len; i++)
		if(arr[i] > max) 
			max = arr[i];
	return max;
}
```
